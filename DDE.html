#3 Linear diff. equation, integrating factor, Bernoulli differential eq.

1. Solve the following diff. eq.
1. $y'+2xy=xe^{-x^2},$
2. $2xy'=y+\frac 3 2 x^2,$
3. $y'+y\cos x = \frac 1 2 \sin 2x$
4. $y'=\frac{y}{\sin x} + \tan \frac{x}{2},$
5. $y'+\frac{xy}{1+x^2}= \frac{\sqrt{1+x^2}}{x^2}.$

2. Solve the following diff. eq.
1. $y'+ \frac{y}{x} = y^2 \ln x$,
2. $xy'+y=y^2 \ln x,$
3. $xy'-y=y^2,$
4. $y'+\frac{y}{x+1} + \frac{(x+1)^3}{2}y^3=0,$
5. $(1-x^2)y' - xy=xy^2.$

#2 vector field, orthogonal trajectories, Euler homogeneous eq.

1. An Example of separable equations.

Solve the differential equations:

1. $y'=x^2y$,
2. $xy'+2y=1$,
3. $(x^2-1)y'=y$,
4. $(x^2+1)y' + y^2 + 1= 0$,
5. $y^2 y'= 1-2x,$
6. $\sin y \cos x y' =\sin x \cos y.$
2. Plot vector field for given d.e.
1. $\frac{dx}{dx} =x$
